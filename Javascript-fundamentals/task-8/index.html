How It Works

This demo uses the browser's window.onhashchange event to detect URL changes and dynamically loads content without page refreshes.

HTML5 CSS3 JavaScript

About this Demo

This SPA demonstrates how to implement client-side routing using hash fragments (#) in the URL.

Key aspects of this implementation:

  • No frameworks or libraries - just vanilla JavaScript
  • State persistence across route changes
  • Smooth transitions between pages
  • Responsive design for all devices
